# State Management Todo App

A simple Todo application built using React Native and Expo.  
This project demonstrates modern state management using Context API and useReducer, along with additional features like drag-and-drop, priority sorting, and dark mode.

## Features

- Add, delete, toggle todo
- Filter (all, active, completed)
- Priority & deadline system
- Smart sorting (urgent first)
- Drag & reorder
- Persistent storage (AsyncStorage)
- Dark mode toggle

## Tech

- React Native (Expo)
- Context API + useReducer
- AsyncStorage
- react-native-draggable-flatlist
- react-native-gesture-handler
- react-native-reanimated
